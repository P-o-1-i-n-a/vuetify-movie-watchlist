# Movie Watchlist (Vuetify)

Приложение, куда можно добавлять фильмы, которые хочется посмотреть.

## Что умеет

- добавлять фильм через форму (название + жанр из выпадающего списка)
- ставить оценку фильму звёздочками
- переключать статус "посмотрено / буду смотреть"
- удалять фильм из списка
- просмотренные фильмы становятся полупрозрачными на карточке

## Использовано

- Vue.js 3 (CDN)
- Vuetify 3 (CDN) — все визуальные компоненты: карточки, диалог, кнопки, свитчер, рейтинг

## Как запустить

Открыть index.html в браузере, нужен интернет (стили и Vue/Vuetify подгружаются с CDN).
